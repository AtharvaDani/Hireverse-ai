# 🚀 HireVerse AI

> AI-Powered Career Intelligence Platform

HireVerse AI is a full-stack AI-powered platform that helps job seekers optimize their resumes, improve ATS scores, identify missing keywords, generate professional cover letters, prepare for interviews, and discover suitable job opportunities.

---

## ✨ Features

- 📄 AI Resume Analysis
- 📊 ATS Resume Score
- 🧬 Resume DNA Fingerprint
- 🔍 Keyword Gap Analysis
- ✍️ AI Resume Rewriter
- 📑 Cover Letter Generator
- 💼 Job Matching
- 🎯 Career Path Recommendations
- 🎤 AI Mock Interview
- ⚡ Fast AI Responses using Groq
- 🤖 Google Gemini Integration
- 📚 Vector Search using pgvector

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/hireverse-ai.git
```

Go to project

```bash
cd hireverse-ai
```

Run Backend

```bash
cd backend
mvn spring-boot:run
```

Run Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file using the provided `.env.example`.

Required variables include:

- Database URL
- Google API Key
- Groq API Key
- Adzuna API Key
- AssemblyAI API Key
